# `docs/`

Holds C4 diagrams, illustrations, and other static assets that the
crate-level rustdoc references. The rustdoc surface itself is the
operational C4 view of the system — see
[`AGENTS.md` § 5.2](../AGENTS.md#52-module-hierarchy--wiki-components).

The architecture spec is normative at the
[UOR-Framework wiki](https://github.com/UOR-Foundation/UOR-Framework/wiki).
Anything here is illustrative only; the wiki is the source of truth.
