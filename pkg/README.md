R4 Prime Router rebased in Rust for full UOR testing.

You must have Ollama installed. You must have gemma4:e2b downloaded for it. You must have Ollama set to allow network connections from your local network.

clone the repo

To Run:
cargo run --release --bin server

GUI:
http://localhost:8000
